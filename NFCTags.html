The below code snippets will help you setup NFC tags based automations in Home Assistant.
Android phone automation
Here is a sample code for a simple NFC tags based automation script for Android phones:
>> Link to download the sample yaml file: Click this link to download NFC tags automation yaml file for Android 

                            alias: "NFC: From webhook"
                            description: ""
                            trigger:
                              - platform: webhook
                                allowed_methods:
                                  - GET
                                local_only: true
                                webhook_id: "-zCH00m2PHOAnNGWipJiKux9R"
                            condition: []
                            action:
                              - service: light.toggle
                                metadata: {}
                                data: {}
                                target:
                                  device_id: z5d19fd60e2e2f9a81c92db7f123456
                            mode: single
